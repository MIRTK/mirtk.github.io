convert-dof2velo¶

Synopsis¶

convert-dof2velo <dofin> <dofout> [options]

Description¶

Reads a 3D transformation and writes a 3D(+t) free-form transformation which is parameterized by a (non-)stationary velocity field and approximates the displacements of the input transformation.

Arguments¶

dofin¶

Input transformation.

dofout¶

Output transformation parameterized by velocities.

Command options¶

-target <image>¶

Target image. Required in case of non-FFD input transformation. (default: none)

-t1 <double>¶

Time point of first frame. (default: temporal origin of FFD/target)

-t2 <double>¶

Time point of second frame. (default: t1 + 1)

-stationary¶

Use stationary velocity field. (default: off)

-int <string>¶

Integration method: RKE1, RKE2, RKBS2, RK4,... (default: RKE2)

-minsteps <int>¶

Minimum number of integration steps. (default: 10)

-maxsteps <int>¶

Maximum number of integration steps. (default: 100)

-steps <int>¶

Number of integration steps. Sets both -minsteps and -maxsteps.

-tol <double>¶

Error tolerance of integration method with adaptive step size. (default: 0.001)

-terms <int>¶

Number of BCH approximation terms (either 2 or 3). (default: 3)

-iters <int>¶

Number of BCH update steps. (default: 8)

-smooth¶

Smooth velocities before each update. (default: off)

-merge¶

Merge global into local displacements. (default: off)

Standard options¶

-v, -verbose [n]¶

Increase/Set verbosity of output messages. (default: 0)

-debug [level]¶

Increase/Set debug level for output of intermediate results. (default: 0)

-version [major.minor]¶

Print version and exit or set version to emulate.

-revision¶

Print revision (or version) number only and exit.

-h, -help¶

Print help and exit.
