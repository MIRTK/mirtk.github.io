em-hard-segmentation¶

Synopsis¶

em-hard-segmentation <N> <atlas1> .. <atlasN> <output> [options]

Description¶

Computes the hard segmentation of the N atlases. Optionally MRF smoothing can be additionally applied.

Input options¶

-mrftimes <number>¶

number of times the mrf term will be applied (default 0)

-mrfweight <double>¶

weight of the mrf term (default 1/3).

-posteriors <post1> .. <postN> write posteriors (useful when MRF is used).¶

Standard options¶

-v, -verbose [n]¶

Increase/Set verbosity of output messages. (default: 0)

-debug [level]¶

Increase/Set debug level for output of intermediate results. (default: 0)

-version [major.minor]¶

Print version and exit or set version to emulate.

-revision¶

Print revision (or version) number only and exit.

-h, -help¶

Print help and exit.
