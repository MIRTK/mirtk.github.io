delete-pointset-attributes¶

Synopsis¶

delete-pointset-attributes <input> [<output>] [options]

Description¶

Delete point data and/or cell data from input point set.

Arguments¶

input¶

Input point set.

output¶

Output point set. (default: input)

Command options¶

-all¶

Remove all point and cell data arrays.

-scalars¶

Delete SCALARS attribute(s).

-vectors¶

Delete VECTORS attribute(s).

-normals¶

Delete NORMALS attribute(s).

-tcoords¶

Delete TCOORDS attribute(s).

-name <name>¶

Name of point/cell data array to remove.

-pointdata <name>|<index>¶

Name of point data array to remove.

-celldata <name>|<index>¶

Name of cell data array to remove.

Standard options¶

-v, -verbose [n]¶

Increase/Set verbosity of output messages. (default: 0)

-debug [level]¶

Increase/Set debug level for output of intermediate results. (default: 0)

-version [major.minor]¶

Print version and exit or set version to emulate.

-revision¶

Print revision (or version) number only and exit.

-h, -help¶

Print help and exit.
