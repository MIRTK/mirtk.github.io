convert-pointset-to-mat¶

Synopsis¶

convert-pointset-to-mat <input> <mat> [options]

Description¶

Saves the points and faces of a vtkPolyData to a MAT file.

Arguments¶

input¶

File name of input dataset.

mat¶

File name of output MAT file.

Command options¶

-points [<var>]¶

Name of points variable. (default: X)

-nopoints¶

Exclude points from MAT file.

-faces [<var>]¶

Name of faces variable. (default: F)

-nofaces¶

Exclude faces from MAT file.

-pointdata <name> [<var>]¶

Include named point data. (default: none)

-celldata <name> [<var>]¶

Include named cell data. (default: none)
