resample-image¶

Synopsis¶

mirtk resample-image <input> <output> [options]

Description¶

Resamples an image on a lattice with specified voxel size.

Arguments¶

input¶

Input image.

output¶

Resampled output image.

Command options¶

-imsize <nx> <ny> <nz>¶

New image size in number of voxels (default: input image size)

-size <dx> <dy> <dz>¶

New voxel size in mm (default: 1 1 1)

-isotropic <m>¶

Resample to isotropic voxel size: m*min(dx, min(dy, dz))

-interp <mode>¶

Interpolation mode (case insensitive): NN, Linear (default), Fast linear, BSpline, Cubic BSpline Fast cubic BSpline, CSpline, SBased, Sinc, Gaussian NN with padding, Linear with padding Fast linear with padding, BSpline with padding Cubic BSpline with padding Fast cubic BSpline with padding, CSpline with padding SBased with padding, Sinc with padding Gaussian with padding -extrap <mode> Extrapolation mode (case insensitive): Default, None, Const (default), NN, Repeat, Mirror ConstWithPeriodicTime -outside <value> Constant outside value extrapolation. (default: 0)

-sigma <value>¶

Sigma value of Gaussian interpolator. (default: 1)

-padding <value>¶

Background padding. (default: none) Only linear interpolation for padding!

Standard options¶

-v -verbose [n]¶

Increase/Set verbosity of output messages. (default: 0)

-debug [level]¶

Increase/Set debug level for output of intermediate results. (default: 0)

-version [major.minor]¶

Print version and exit or set version to emulate.

-revision¶

Print revision (or version) number only and exit.

-h -help¶

Print help and exit.
