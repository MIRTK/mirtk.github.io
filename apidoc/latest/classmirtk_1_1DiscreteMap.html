Detailed Description

Volumetric map defined at mesh nodes

This volumetric map is defined by its values at discrete mesh nodes. Values between nodes are interpolated using the weights of the respective mesh cell which this point belongs to. If the point is not contained in any mesh cell, the point is mapped to a constant outside value.

A tetrahedral mesh is commonly used to parameterize a discrete volumetric map.
