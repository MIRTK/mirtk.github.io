49 /**
50  * Abstract base class for general transformations.
51  *
52  * This is the abstract base class which defines a common interface for all
53  * transformations. Each derived class has to implement at least the abstract
54  * methods and some of the virtual ones. Most other methods call these virtual
55  * methods and should not be required to be overwritten in subclasses.
56  *
57  * The second time argument to the interface methods corresponds to the time
58  * of the untransformed source image. It is only considered by some 3D+t
59  * transformations, in particular those which parameterize the transformation
60  * using a non-stationary velocity field.
61  */

752 // -----------------------------------------------------------------------------
753 inline bool HaveSameDOFs(const Transformation *t1, const Transformation *t2)
754 {
755  // If the virtual HasSameDOFsAs function was not overriden by the specialized
756  // type of one transformation, it is assumed that it uses the _Param memory to
757  // store its transformation parameters. However, the other transformation
758  // might be a specialized type which does not make use of it. In this case,
759  // even if the two _Param pointers do not reference the same memory, the two
760  // transformations may still use the same parameters because the other
761  // transformation directly or indirectly wraps this transformation. Therefore,
762  // the check with the transformations exchanged and the boolean OR (not AND).
763  return t1->HasSameDOFsAs(t2) || t2->HasSameDOFsAs(t1);
764 }
