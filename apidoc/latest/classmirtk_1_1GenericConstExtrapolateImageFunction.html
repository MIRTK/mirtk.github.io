Detailed Description

template<class TImage>
class mirtk::GenericConstExtrapolateImageFunction< TImage >

Constant extrapolation (i.e., padding) of generic image

This discrete image extrapolation function assumes a constant image value for image grid points outside the discrete domain on which the discrete image function is defined.
