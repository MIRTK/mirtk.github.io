318  /// Add change (i.e., scaled gradient) to each deformable surface parameter
319  ///

322  /// energy function.
323  ///

410  /// Enforce hard constraints on surface model deformation
411  ///
412  /// This function clamps a nodes' displacement vector (velocity times \f$\delta t\f$),
413  /// if otherwise the hard constraints of the deformable surface model would
414  /// be violated. Common hard constraints are non-self-intersection and a
415  /// maximum total node displacement. If the surface model is deformed by
416  /// a parametric transformation, this function does nothing as hard constraints
417  /// can only be enforced during the optimization when the parameters of the
418  /// deformable surface model are the positions of the individual surface nodes.
419  ///
420  /// \param[inout] dx (Scaled) gradient of objective function.
421  virtual void EnforceHardConstraints(double *dx) const;
