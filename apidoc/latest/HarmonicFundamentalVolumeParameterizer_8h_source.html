32 /**
33  * Harmonic volumetric map using the method of fundamental solutions (MFS)
34  *
35  * This class finds a harmonic volumetric map which maps each point of the
36  * input volume to a point in the target domain using the method of fundamental
37  * solutions (MFS).
38  *
39  * - Li et al. (2009). Meshless harmonic volumetric mapping using fundamental solution methods.
40  * IEEE Transactions on Automation Science and Engineering, 6(3), 409–422.
41  * - Li et al. (2010). Feature-aligned harmonic volumetric mapping using MFS.
42  * Computers and Graphics (Pergamon), 34(3), 242–251.
43  */
