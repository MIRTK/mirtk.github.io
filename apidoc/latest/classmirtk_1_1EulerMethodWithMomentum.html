Detailed Description

Minimizes deformable surface model using Euler's method with momentum

This method is used by the FreeSurfer tools:

Fischl et al., Cortical Surface-Based Analysis II: Inflation, Flattening, and a Surface-Based Coordinate System. NeuroImage, 9(2), 195â€“207 (1999).
