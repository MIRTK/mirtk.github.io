31 /**
32  * Cubic B-spline interpolation of generic image
33  *
34  * Note that the coefficient, i.e., input voxel type may also be a vector
35  * type such as Vector3D in particular. This is required by the B-spline
36  * free-form transformations. If the input image contains the cubic B-spline
37  * interpolation coefficients already, make sure that the VoxelType template
38  * argument matches the voxel type of the input image to avoid an unnecessary
39  * copy of the input image. This image function will use the memory of the
40  * input image directly to access the coefficients in this case.
41  */
42 template <class TImage>
