Detailed Description

Class for affine transformations.

This class defines and implements affine transformations. In addition to the rigid body transformation parameters, affine transformations are parameterized by three scaling and and six skewing parameters. The three scaling parameters define the scaling along the axis of the coordinate transformations. The six skewing parameters define the skewing angles in different planes. Note that the six skewing parameters are not independent.
