100  /// Point data array used to adapt the edge length range for each node
101  ///
102  /// The scalar point data values are rescaled linearly to [0, 1] after
103  /// clamping the point data range to the 5th and 95th percentile range.
104  /// The rescaled value is then plugged into a logistic function which
105  /// determines the linear interpolation weights of the global
106  /// _MinEdgeLength and _MaxEdgeLength range. This obtains an individual
107  /// edge length range for each point. The desired edge length range of a
108  /// given edge is then the mean of the minimum/maximum edge length of the
109  /// two end points of the edge.
110  mirtkPublicAttributeMacro(vtkSmartPointer<vtkDataArray>, AdaptiveEdgeLengthArray);

230  /// Check connectivity of edge neighbors
231  ///
232  /// If other vertex of edge neighbor triangle has connectivity three,
233  /// the three triangles adjacent to it are replaced by the union of these
234  /// triangle and the melting operation can be performed. Otherwise, if
235  /// more than one node with connectivity three is found which is adjacent
236  /// to the edge corners, melting the (triangle) edge would cause degeneration
237  /// of adjacent triangles.
238  ///
239  /// @return ID of other vertex of edge neighbor or -1 if not unique or if
