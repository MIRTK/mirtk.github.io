Detailed Description

template<class TVoxel>
class mirtk::GaussianBlurring< TVoxel >

Class for Gaussian blurring of images

This class defines and implements the Gaussian blurring of images. The blurring is implemented by three successive 1D convolutions with a 1D Gaussian kernel.

By default, if one isotropic Gaussian standard deviation is specified, the first three dimensions of the image are blurred only. Otherwise, the 1D convolution with a 1D Gaussian kernel is performed only for dimensions of more than one voxel size and for which a non-zero standard deviation for the Gaussian kernel has been set.
