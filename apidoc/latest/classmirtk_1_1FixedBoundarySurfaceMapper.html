Detailed Description

Base class of solvers computing a surface map given a fixed boundary map

Solvers of this type compute a surface map value for each interior point of the non-closed input surface mesh given a fixed map value for each point on at least one of the boundary segments. Additional either weak or hard constraints at interior points may also be considered.
