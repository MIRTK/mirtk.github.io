174  /// Add change (i.e., scaled gradient) to each transformation parameter
175  ///
176  /// This function updates each parameter of the registration energy function
177  /// given a vector of desired changes, i.e., the computed gradient of the
178  /// energy function w.r.t. these parameters or a desired change computed
179  /// otherwise such as through matching patches (in a very loose sense an
180  /// approximate "gradient") or a discrete assignment (c.f. MRF registration).
181  /// It moreover triggers an update of the internal state of each energy term
182  /// which depends on the current transformation. A data similarity term,
183  /// for example, will update its moving (transformed) input data.
184  ///
185  /// \note How this change is applied to the transformation parameters depends
186  /// on the particular underlying transformation model. Often \p dx is
187  /// simply added to \c x, the current transformation parameters. Some
188  /// transformations may compute a slightly different update from \p dx,
189  /// however, such as converting displacements into velocities first.
190  ///
191  /// \param[in] dx Change of each function parameter (DoF) as computed by the
192  /// Gradient member function and scaled by a chosen step length.
193  ///
194  /// \returns Maximum change of transformation parameter.
195  virtual double Step(double *dx);
196 
197  /// Update internal state after change of parameters
198  virtual void Update(bool = true);

200  /// Update energy function after convergence

204  /// the new FRE term.

206  /// \returns Whether the energy function has changed.

230  /// Evaluate gradient of registration energy
231  ///
232  /// Note that this gradient need not necessarily correspond to the analytic
233  /// gradient of the energy function. It can also be a desired change of each
234  /// function parameter computed otherwise such as through a local patch
235  /// match. The only requirement is that the change will increase the energy
236  /// function value such that successively walking in the direction of the
237  /// computed "gradient", a (local) maximum of the registration energy will be
238  /// be reached.
239  ///
240  /// \param[in] dx Gradient of registration energy function.
241  /// \param[in] step Step length for finite differences.
242  /// \param[out] sgn_chg Whether function parameter value is allowed to
243  /// change sign when stepping along the computed gradient.
244  void Gradient(double *dx, double step = .0, bool *sgn_chg = NULL);
245 
246  /// Compute norm of gradient of registration energy
